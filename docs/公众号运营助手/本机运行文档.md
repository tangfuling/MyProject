# 公众号数据运营助手 - 本机运行文档

## 1. 目录与环境

项目目录：`/Users/ke/Documents/Mine/Works/MyProject/gzh-react`

要求：
- Node.js >= 18（当前机器是 `v22.22.1`）
- npm >= 9（当前机器是 `10.9.4`）

## 2. 安装依赖

```bash
cd /Users/ke/Documents/Mine/Works/MyProject/gzh-react
npm install
```

## 3. 配置本地 API 地址

`gzh-react` 当前通过 `VITE_API_BASE_URL` 读取后端地址；如果不配置，会默认打到 `https://api.example.com`。

建议新建 `.env.development`：

```bash
cat > .env.development <<'ENV'
VITE_API_BASE_URL=http://127.0.0.1:8081
ENV
```

如果你本地后端不是 `8081`，改成你的实际地址即可。

## 4. 启动开发环境

```bash
npm run dev
```

默认会启动 Vite 开发服务器（通常是 `http://localhost:5173`）。

## 5. 本机构建与预览

构建生产包：

```bash
npm run build
```

构建产物在 `gzh-react/dist`。

本机按生产模式预览：

```bash
npm run preview
```

## 6. 常用命令速查

```bash
# 开发
npm run dev

# 打包
npm run build

# 预览生产包
npm run preview
```
